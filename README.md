# 📖 Overview

OmniSupport AI is a fully customizable AI chatbot platform that can be embedded into any website using a single script.

The platform allows businesses to:

✅ Provide instant customer support

✅ Answer FAQs automatically

✅ Generate leads

✅ Capture customer information

✅ Train AI on custom company data

✅ Support multiple websites

✅ Manage conversations from a central dashboard

✅ Deploy with a single line of code

# 🌐 Website Integration

Simply add:

```html
<script src="https://cdn.omnisupport.ai/widget.js"></script>

<script>
OmniSupport.init({
    chatbotId: "YOUR_CHATBOT_ID"
});
</script>
```

And the chatbot instantly appears on the website.
